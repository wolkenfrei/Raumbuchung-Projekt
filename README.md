# Raumbuchung-Projekt
Buchungssystem mit Python & Streamlit

Praxisnahes Projekt zum Aufbau von Programmierkenntnissen.
🌐 Ziel der Anwendung

Eine einfache Web-Anwendung zur Buchung von Räumen und Sitzplätzen.

Die Anwendung soll im Browser laufen und sowohl auf dem PC als auch auf dem Handy nutzbar sein.
🎯 Projektidee

Eine Streamlit-App, mit der Benutzer folgendes machen können:

* einen Raum auswählen können
* einen Platz auswählen können
* sehen können, ob ein Platz frei oder gebucht ist
* einen freien Platz buchen können
* einen Platz nach Nutzung wieder freigeben können

Zusätzlich soll es eine einfache Feedback-Funktion geben sowie eine Admin-Sicht.
💡 Warum dieses Projekt

Dieses Projekt dient als Lernprojekt, um:

* Python von Grund auf besser zu verstehen
* strukturierten Code zu schreiben
* UI-Logik mit Streamlit umzusetzen
* ein praxisnahes System Schritt für Schritt aufzubauen
* jeden Teil des Codes selbst erklären zu können
* einfach Spaß und schnelle Erfolge sehen zu können

Der Fokus liegt auf:

* Verständlichkeit statt Komplexität
* klarer Struktur im Code (Block-Prinzip)
* kleinen, nachvollziehbaren Schritten
  ⚙️ Warum Streamlit

Streamlit ermöglicht die Entwicklung von Web-Anwendungen direkt in Python.

Vorteile:

* keine HTML-Kenntnisse notwendig
* keine CSS-Kenntnisse notwendig
* schnelle Umsetzung von Ideen
* sofort sichtbare Ergebnisse im Browser
* läuft auf PC und mobilen Geräten
  🧩 Geplante Funktionen

Die Anwendung wird schrittweise aufgebaut.

Geplant sind unter anderem:

* Auswahl von Räumen und Plätzen
* Anzeige von freien und gebuchten Plätzen
* Buchung eines Platzes
* Freigabe eines Platzes nach Nutzung
* Feedback nach Freigabe
* Admin-Bereich zur Übersicht aller Buchungen
  🏗 Projektstruktur

Der Code wird bewusst in logische Blöcke unterteilt, um die Lesbarkeit zu verbessern.

Geplante Struktur:

Block 0 – Imports
Block 1 – Page Setup und Layout
Block 2 – Daten (Räume und Plätze)
Block 3 – Auswahl im UI
Block 4 – Buchungslogik
Block 5 – Ausgabe und Rückmeldungen

Diese Struktur soll helfen, den Code klar zu verstehen und später selbst erklären zu können.
